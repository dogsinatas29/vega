# 1. Runtime Scenario

## Scenario 1 — Backup Operation

User input:
"backup server A data to backup server"

Flow:
1. Natural language command received
2. Intent Resolver analyzes intent
3. Command template selected
4. AI generates command options
5. Command AST created
6. Virtual execution predicts impact
7. Risk evaluation validates safety
8. SSH execution performed
9. Decision lineage recorded
10. Report generated

Example command:
rclone sync /data backup:/data --progress --checksum

Simulation:
Transfer: 10GB
Deleted files: 32
Estimated duration: 3 minutes
Risk score: 12
Result: Execution allowed

→ Intent Resolver

→ Command AST

→ Risk Evaluation
→ Remote Execution Layer
→ Decision Lineage
→ Report Generator

---

## Core Modules

Natural Language Interface
Receives operator instructions.

Intent Resolver
Converts natural language to structured intent.

Command Template Builder
Creates deterministic command skeleton.

AI Option Generator
Generates command options only.

Command AST
Structured command representation.

Virtual Execution Engine
Simulates command effects before execution.

Risk Evaluation Engine
Calculates safety score.

Remote Execution Layer
Executes commands via SSH provider.

Decision Lineage
Stores full execution trace.

Report Generator
Creates structured execution reports.
